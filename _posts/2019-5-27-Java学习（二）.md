# Spring入门

#### 框架概述
Spring框架基于POJO（简单java对象），总体由：
* Core，IoC容器，Framework工具类
* JEE服务集成，JMX，JMS等
* AOP，Spring AOP，AspectJ集成
* DAO，Spring JDBC，事务管理
* ORM，Hibernate，iBATIS，JPA，Toplink，JDO
* Web，SpringMVC等等
各种服务实现组成了Spring框架的核心骨架。
